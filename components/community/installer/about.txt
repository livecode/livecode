LiveCode

by LiveCode

LiveCode is the open source app creation platform that empowers everyone to create apps.

It provides the easiest path to coding and app creation for those starting out, learning to program or creating their first app. For experienced professionals, LiveCode is the fastest route to creating your next #1 app, utility or powerful new business system.

LiveCode lets you:
- Create your user interface by drag dropping objects onto a canvas
- Use an English-like language to add logic to your app
- Include graphics, multimedia or animation
- Process text and data using natural language
- Edit your app while it is running live
- Write your app once then deploy it to iOS, Android, Windows, Mac or Linux
- Create server applications using the same language

Founded in 1997, LiveCode are the makers of LiveCode. LiveCode was formerly known as Revolution then RunRev. LiveCode is based in Edinburgh, Scotland.
